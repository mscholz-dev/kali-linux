# Ajout de paquet : chromium

[↩️ Retour](./README.md)

## Description

Avoir un navigateur web basé sur chrome

## 1. Mise à jour du paquet apt-get

```bash
apt-get update
apt-get upgrade
```

## 2. Installation

```bash
apt install chromium
```

## 3. Lancer Chromium avec un utilisateur temporaire pour la session

```bash
chromium --temp-profile
```
