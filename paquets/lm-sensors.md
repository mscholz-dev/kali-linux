# Ajout de paquet : lm-sensors

[↩️ Retour](./README.md)

## Description

Permet de gérer la température, la tension, la vitesse des ventilateurs et l'humidité du système.

## 1. Installation

```bash
apt install lm-sensors
```

## 2. Configuration

```bash
sensors-detect
```

## 3. Afficher les données du matériel

```bash
sensors
```
